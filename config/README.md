# Конфигурация полей заказа

Этот файл содержит единую конфигурацию полей заказа для всех страниц калькулятора.

## Структура

- `orderFieldsConfig` - объект с конфигурациями для каждой страницы
- `getOrderFieldsConfig(key)` - функция для получения конфигурации по ключу страницы
- `getPageConfigKeyFromRoute(routePath)` - функция для получения ключа конфигурации по маршруту

## Использование

В компонентах страниц используйте:

```typescript
import { getOrderFieldsConfig } from "~/config/order-fields-config";

const fields = reactive<OrderField[]>(getOrderFieldsConfig("tracing"));
```

## Добавление новой страницы

1. Добавьте новый ключ в тип `PageConfigKey`
2. Добавьте конфигурацию в объект `orderFieldsConfig`
3. При необходимости добавьте маппинг в функцию `getPageConfigKeyFromRoute`

## Интеграция с админ-панелью

В дальнейшем этот файл можно будет заменить на загрузку конфигурации из API или базы данных. Структура данных остается той же, меняется только источник данных.
